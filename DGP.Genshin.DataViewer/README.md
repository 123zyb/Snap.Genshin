## 教程

* 选择包含Excel与TextMap文件夹的文件夹
* Excel文件夹与TextMap文件夹分别包含配置信息与文字映射信息
* 选择文件夹并读取成功后，选择一种语言，如 TextZHS
* 在左侧的列表中选择想要浏览的文件
* 开始浏览/分析你的数据

<font color="red">注：**本App只负责重新映射Json文件中的某些特定属性，不具备解包能力**</font>
